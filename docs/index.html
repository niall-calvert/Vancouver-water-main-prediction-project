About this application

This map shows water main pipe segments that the machine learning model classified as predicted leaks for the next year.

Predicted probability

The leak probability is the model's estimated chance that a pipe will leak in the next year. For example, 82% means the model estimated an 82% next-year leak probability for that pipe-year row.

Predicted leak

A predicted leak is a pipe whose predicted probability is above the model's classification threshold. In this project, the threshold is approximately 82%.

Risk rank

Risk rank orders pipes from highest to lowest predicted probability. A lower rank means the pipe is considered higher risk by the model.

Pipe age

Pipe age is calculated from the pipe's installation year and the analysis year. The slider filters out pipes younger than the selected age.

Material and lining

Material and lining material describe the physical pipe construction. These fields are included because different pipe types may have different leak patterns.

Water main type

Water main type separates distribution and transmission mains when that source information is available.

Important limitation

This model is intended for risk screening and prioritization. It should not be treated as a guaranteed prediction that a specific pipe will or will not leak.
